# 🎓 Scholarisk AI

**AI-powered Student Risk Intelligence System**

Scholarisk is not just a burnout predictor — it is a multi-layer system that analyzes student behavior, detects hidden academic risks, explains the causes, and provides actionable improvement strategies.

---

## 🚀 Key Features

### 🔍 Burnout Prediction (ML-Based)
Uses Logistic Regression to estimate burnout probability based on lifestyle factors.

### 📊 Risk Score Engine
Calculates an overall risk score (0–100) combining multiple behavioral indicators.

### 🧠 Explainability Engine
Provides clear reasons behind predictions (Why burnout risk is high/low).

### 🧬 Student Type Detection
Classifies users into behavioral categories like:
- Overloaded Achiever  
- Distracted Explorer  
- Focused Performer  

### 🚨 Priority Action System
Identifies the most critical issue to fix first.

### 🔮 Future Prediction
Estimates potential outcomes if current habits continue.

### 🔄 Improvement Simulation
Shows how risk changes if habits improve.

### 📅 Personalized Recovery Plan
Generates a daily plan based on user inputs.

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- Scikit-learn (Logistic Regression)  
- Streamlit  

---

## 💡 Problem Statement

Students often face burnout and poor performance without understanding the root cause.  
Most existing systems only provide predictions without explanations or solutions.

---

## ✅ Solution Approach

Scholarisk combines:
- Machine Learning (prediction)
- Rule-based logic (behavior analysis)
- Explainable outputs (reasoning)

to create a complete **decision-support system**.
