# 🎓 Scholarisk AI

AI-powered Student Risk Intelligence System that detects burnout, analyzes behavior, and provides personalized interventions.

## 🚀 Features
- Burnout Prediction
- Risk Score Analysis
- Personality Detection
- Explainability (Why result)
- Priority Action System
- Future Prediction
- Daily Recovery Plan

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Streamlit

## ▶️ Run
pip install pandas scikit-learn streamlit  
python model.py  
python -m streamlit run app.py
